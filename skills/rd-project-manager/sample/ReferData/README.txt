ReferData — 專案參考資料資料夾
=====================================

此資料夾用來存放開發過程中的所有參考資料。
Place all reference materials for this project in this folder.

建議分類 / Suggested categories:
  - 參考文件 (Documents)   : PDF, Word, TXT from customers or vendors
  - 參考照片 (Photos)      : Screenshots, UI mockups, hardware photos, diagrams
  - 參考 SPEC (Specs)      : Competitor specs, standards, prior-version specs
  - 其他 (Others)          : Email threads, meeting notes, datasheets

注意 / Notes:
  - 本資料夾內容由開發者自行管理，AI skill 不會修改或刪除此資料夾內的任何檔案。
  - This folder is developer-managed. The rd-project-manager skill will
    never modify or delete files here.
  - You may reference files here in IssueList or SPEC by filename.
    Example: "See ReferData\customer_requirements_v2.pdf"
